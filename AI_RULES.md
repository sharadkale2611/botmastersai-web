Always:

* Use TypeScript.
* Use functional components.
* Use TailwindCSS.
* Use App Router.
* Use mobile-first design.
* Export default components.
* Avoid inline styles.
* Keep components reusable.
* Use modern SaaS design.

Folder rules:

* Never create app/components.
* Never create duplicate component folders.
* All components must be inside src/components.
* Use alias imports:

@/components/home/...

* page.tsx should only compose sections.
* Do not place UI code directly inside page.tsx.
* Every section must have its own component.

Naming rules:

HeroSection.tsx

AboutSection.tsx

ServicesSection.tsx

FAQSection.tsx

Footer.tsx

Navbar.tsx

Never use index.tsx files.

Never use barrel exports.
