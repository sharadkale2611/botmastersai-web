## BotMasters AI Project Instructions

Before making any changes, read:

* AGENTS.md
* AI_RULES.md
* ARCHITECTURE.md
* CONTENT.md
* TASKS.md
* COMPONENT_TREE.md
* DEVELOPMENT_WORKFLOW.md

Follow them strictly.

Never:

* Create app/components.
* Create duplicate folders.
* Move completed components without reason.
* Place UI directly inside page.tsx.
* Change colors or branding.

Always:

* Use src/components.
* Use alias imports.
* Update imports after moving files.
* Preserve completed components.
* Maintain modern SaaS styling.
* Use logo.png from the public folder whenever a logo is required.

After changes, provide a summary of:

* Files modified
* Files added
* Files removed
* Imports updated
* Any issues found
